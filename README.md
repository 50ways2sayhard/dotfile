# 50ways2sayhard's dotfiles

## Installation

### Install stow

For macOS user,

```shell
brew install stow
```

For Arch Linux user,

```shell
pacman -S stow
```

### Install dotfile

Manually,

```shell
git clone --recursive https://github.com/50ways2sayhard/dotfile
```

### Create links

```shell
cd dotfile
stow <module>
```
